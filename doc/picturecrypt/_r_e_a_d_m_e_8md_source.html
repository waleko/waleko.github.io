4 ## External usage
5 ModelPC class can be used externally (without UI)
6 ```
7 #include <modelpc.h>
8 #include <QByteArray>
9 #include <QImage>
10 
11 ...
12 
13 ModelPC * model = new ModelPC(ver);
14 // ver is version of the app, used to check the data structure version
15 // ver is type long and is calculated as if version is "x.y.z" => ver = x * 65536 + y * 256 + z
16 // Default parameter is 2^17 (2.0.0)
17 
18 // Connecting signals

20 // Essential ones
